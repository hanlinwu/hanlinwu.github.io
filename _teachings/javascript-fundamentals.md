## Course Overview

This course introduces JavaScript programming for modern web development. Students will learn the language fundamentals and practical web development skills.

## Topics Covered

- JavaScript basics: variables, types, operators
- Functions and scope
- DOM manipulation
- Asynchronous programming: callbacks, promises, async/await
- Modern ES6+ features
- Introduction to front-end frameworks

## Code Examples

```javascript
// Fetch data from an API
async function fetchData(url) {
  const response = await fetch(url);
  const data = await response.json();
  return data;
}
```
