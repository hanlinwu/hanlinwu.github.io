# Hanlin Wu (吴瀚霖) — Vue (No SSG)

This branch hosts a native Vue-powered version of the site without a static site generator.

## How to run
- Open `index.html` directly in a browser, or
- Serve the repository root with any static file server.

## Notes
- Styling is reused from the original theme assets (`/css`, `/dist`, `/media`).
- Content lives in `js/vue-app.js` and is rendered by Vue at runtime.
